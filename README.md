# Rust Course for C++ and Python Programmers

This repo is a Rust course with 20 main parts plus one supplemental lesson. Each part has:

- a lesson in [`lessons/`](./lessons)
- a matching Cargo exercise in [`exercises/`](./exercises)

Work through each lesson, complete the exercise, and check it off when you're done.

## Quick Start

```bash
cd exercises/01-setup-and-first-program
cargo run
cargo test
```

Use `cargo fmt` and `cargo clippy` when helpful. Exercise instructions live in code comments and TODOs.
